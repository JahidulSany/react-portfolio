# 🚀 React Portfolio

A dynamic and responsive React-based portfolio to showcase skills and accomplishments. The portfolio demonstrates understanding of React fundamentals, responsive design, and component-based architecture. 

## 🌐 Repository and Live Link

GitHub Repo: https://github.com/JahidulSany/react-portfolio
Live Link: https://react-portfolio-13mu.onrender.com/

---

# ✨ Features

- Fetch GitHub repositories using the **GitHub API**
- Display projects in a **responsive gallery**
- Show repository statistics (repo name, description, language)
- Clean and simple **React component structure**
- Built using **React + Axios + HTML + CSS**

# ⚙️ Installation

## 1. Clone the repository

```
git clone https://github.com/JahidulSany/react-portfolio.git
```

## 2. Navigate to the project folder

```
cd react-portfolio
```

## 3. Install dependencies

```
npm install
```

## 4. Install Axios

```
npm install axios
```

## 5. Start the development server

If using Vite:

```
npm run dev
```

# 🔗 GitHub API Example

The portfolio fetches repositories using:

```
https://api.github.com/users/JahidulSany/repos
```

Example Axios request:

```javascript
const response = await axios.get(
  "https://api.github.com/users/JahidulSany/repos?per_page=100"
);
```
